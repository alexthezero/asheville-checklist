# Asheville NC Checklist Web App

A static GitHub Pages-ready web app that lists things to do in Asheville, North Carolina and lets the user check off completed places.

Checklist progress saves locally in the browser using `localStorage`.

## Files

- `index.html`
- `style.css`
- `app.js`
- `README.md`

## Features

- Asheville things-to-do list
- Checkbox for every place
- Saves progress to local browser memory
- Search box
- Category filter
- Detail cards
- Google Maps links
- Official/info links
- Copy checked list button
- Reset checklist button
- Mobile-friendly layout

## Publish on GitHub Pages

1. Open this repository on GitHub.
2. Go to **Settings**.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select:
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your site should publish at:

```text
https://alexthezero.github.io/asheville-checklist/
```

## Notes

The app has no database. It saves checklist progress only on the current device/browser. If you open the same site on another phone or computer, it will have its own separate checklist state.
